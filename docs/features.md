# 核心功能清单

## 1. 应用管理

### 应用注册
- 创建应用实例，设置应用名称、标识符
- 配置平台类型（win32/darwin）和架构（x64/arm64）

### 应用查询
- 查看所有已注册应用
- 支持按平台、架构筛选

### 应用删除
- 删除应用及其版本数据

## 2. 版本管理

### 版本发布
- 创建新版本，设置 appVersion 和 rendererVersion
- 添加版本描述和更新日志

### 版本查询
- 查看应用所有版本
- 查看版本详细信息

### 版本删除
- 删除指定版本

## 3. 文件上传

### 全量包上传
- 上传安装程序（.exe/.dmg/.apk）
- 自动计算 SHA256 哈希值

### 增量包上传
- 上传热更新压缩包（.zip）
- 自动计算 SHA256 哈希值

### 元数据生成
- 自动生成 electron-updater 兼容的版本元数据文件

## 4. 版本检查接口

### 更新检查
- GET /check 接口
- 查询参数：appVersion, rendererVersion, platform, arch
- 返回更新类型：none/hot/installer
- 返回新版本号、下载地址、SHA256

### 更新策略
- 优先推荐热更新
- 自动版本对比
- 支持强制更新

## 5. 存储分发

### 云存储集成
- 集成 OSS/OBS 存储服务
- 自动管理文件路径

### 预签名 URL
- 生成临时下载链接
- 确保访问安全

### CDN 加速
- 配置 CDN 域名
- 自动同步到 CDN 节点

## 6. 安全控制

### 认证授权
- 用户登录认证
- 基础权限控制

### 文件校验
- SHA256 完整性校验
- 支持文件签名验证
